スクリプトを付与した対象にスポットライトのようなものを表示する。  
懐中電灯で暗闇を照らした時のような円形の見た目となる。  

# スポットライト_AKIRA.anm2

1つしかスポットライトのようなものを付与できないが、「反転」「元のサイズに合わせる」などのチェックボックスが使える。  
位置や大きさを調整可能。  

# スポットライトv2_AKIRA.anm2

複数のスポットライトのようなものを付与できるようになったが、「反転」「元のサイズに合わせる」などのチェックボックスが使えなくなっている。  
「反転」「元のサイズに合わせる」はこちらのスクリプトで正常に動作しないようであるため、これらの機能を使えないようにしています。  
位置や大きさを調整可能。

名称は被らないように、との判断から"_AKIRA"を付けています。  
それ以外の意図はなく、こだわりなどはないため、適宜変更していただいた構いません。  

二次配布可、改変可、クレジット不要
